# PR review instructions (this repo)

This template repository uses the rubric at [`templates/docs/pr_review_instructions.md`](../templates/docs/pr_review_instructions.md) as its own canonical PR review rubric — the file that ships into consuming projects IS the rubric for reviewing this repo. No separate copy: one source of truth, no drift.

When a reviewer is pointed at "`docs/pr_review_instructions.md`" per `PROJECT_STARTER.md §2.7`, this pointer file resolves the path on this meta-repo. In consuming projects, `docs/pr_review_instructions.md` IS the rubric (the template file gets copied there at bootstrap).
